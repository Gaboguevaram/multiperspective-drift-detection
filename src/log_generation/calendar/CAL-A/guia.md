# CAL-A — Reducción de calendario con recurso de control (2 recursos)

Dos recursos: Marta y Carlos. En el log base ambos trabajan de lunes a domingo. Tras el cambio, Marta reduce su jornada a lunes-viernes mientras Carlos permanece igual (lunes-domingo).

- Tipo de cambio: reducción de la jornada de un recurso; el otro actúa como control.
- Métrica que reacciona: soporte inverso de Marta cae; el de Carlos permanece plano.
- Como el soporte se calcula por recurso, el cambio de Marta no se diluye con el de Carlos: demuestra el aislamiento entre recursos.
- Caso usado para CAL-A en la batería de logs simples.

# CAL-B — Ampliación de calendario con recurso de control (2 recursos)

Dos recursos: Marta y Carlos. En el log base Marta trabaja de lunes a viernes y Carlos de lunes a domingo. Tras el cambio, Marta amplía su jornada a lunes-domingo mientras Carlos permanece igual (lunes-domingo).

- Tipo de cambio: ampliación de la jornada de un recurso; el otro actúa como control.
- Métrica que reacciona: soporte de Marta cae; el de Carlos permanece plano.
- Como el soporte se calcula por recurso, el cambio de Marta no se diluye con el de Carlos: demuestra el aislamiento entre recursos.
- Caso usado para CAL-B en la batería de logs simples.
